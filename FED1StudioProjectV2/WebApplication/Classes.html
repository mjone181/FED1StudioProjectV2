Our Courses:

This is a page for all the various classes that we teach in our institution.

Introduction to Game Design

This course provides a basic introduction to the processes entailed in designing a game, such as taking ideas and turning them into physical plans/outlines to work off of.

Introduction to Game Programming

This course provides a basic introduction to the processes entailed in programming a game, such as using the engine to physically build prototypes based on specific concepts core to a game's experience.

Introduction to Game Engines

This course provides a basic introduction to the processes entailed in making a game engine for programmers and designers to work off of, as well as what goes into the backbone of how most games work/function.

Game Business/Marketing

This course provides an understanding of various marketing techniques/strategies used in the game industry to sell various software products.

Advanced Game Design

This course provides an understanding of advanced design concepts used in most game development studios, as well as actual techniques used throughout the industry.

Advanced Game Programming

This course provides an understanding of advanced programming techniques used in the games industry, as well as techniques that can transfer over outside of game development itself.
